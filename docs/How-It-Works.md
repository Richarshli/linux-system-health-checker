# How It Works

This project uses basic Linux commands to collect important system health information.

---

## Commands Used

### date

Displays the current system date and time.

---

### uptime

Shows how long the system has been running and the current load average.

---

### df -h

Displays disk usage in a human-readable format.

---

### free -h

Displays memory usage.

---

### lscpu

Displays CPU architecture and processor information.

---

### who

Lists users currently logged into the system.

---

### whoami

Displays the current user.

---

### hostname

Displays the computer's hostname.

---

### ip addr show

Displays IP address information for network interfaces.

---

### ps aux

Displays currently running processes sorted by memory usage.

---

## Purpose

The goal of this project is to demonstrate Linux command-line skills, basic Bash scripting, and system monitoring techniques commonly used by Linux administrators, help desk technicians, cybersecurity analysts, and IT support professionals.
